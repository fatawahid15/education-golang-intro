# Golang Introduction Course

Welcome to this introductory course on the Go programming language. This course is designed for beginners who want to learn the basics of Go.

## Prerequisites

Before you begin, you should have the following installed:

-   [Go (version 1.18 or higher)](https://go.dev/dl/)
-   [Git](https://git-scm.com/downloads)
-   A code editor of your choice (e.g., [Visual Studio Code](https://code.visualstudio.com/))

## Course Sections

This course is divided into several sections, each covering a fundamental concept of Go.

| #  | Section                                       | Description                                                                 |
|----|-----------------------------------------------|-----------------------------------------------------------------------------|
| 01 | [Hello, World!](./01-hello-world/)            | The classic first program to get you started with Go's syntax.              |
| 02 | [Imports](./02-imports/)                      | Learn how to import and use packages from Go's standard library.            |
| 03 | [Naming Conventions](./03-naming-conventions/) | Understand the idiomatic way to name variables, functions, and types in Go. |
| 04 | [Variables](./04-variables/)                  | Cover the basics of variable declaration, initialization, and scope.        |

## How to Use This Repository

1.  Clone the repository to your local machine.
2.  Navigate to a section's directory (e.g., `cd 01-hello-world`).
3.  Read the `README.md` file in that directory for an explanation of the concept.
4.  Run the `main.go` file using `go run main.go` to see the code in action.

Happy coding!
